## Descripción del Proyecto
StudyMate AI es un asistente de estudio que ayuda a estudiantes a comprender y repasar material académico de forma más eficiente. El usuario sube un documento o apunte de clase, y el sistema utiliza inteligencia artificial para generar automáticamente un resumen claro del contenido junto con un set de preguntas tipo quiz para autoevaluación. El objetivo es reducir el tiempo de preparación de exámenes y reforzar el aprendizaje activo mediante retroalimentación inmediata generada por IA.
## Stack Tecnológico
* **Frontend:** Angular
* **Backend:** Java + Spring Boot
* **Base de Datos:** PostgreSQL
* **IA:** Spring AI + Modelos de Lenguaje
